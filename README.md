# 🎓 Student Study Tracker

A web application to track student study hours and progress.

## 🛠️ Tech Stack
- HTML (Frontend Structure)
- CSS (Frontend Styling)
- JavaScript (Frontend Logic)
- Node.js + Express (Backend)
- MySQL (Database)

## ✨ Features
- User Register & Login
- Add Subjects
- Log Study Hours Daily
- Progress Dashboard
- Weekly Study Goals
- To-Do List
